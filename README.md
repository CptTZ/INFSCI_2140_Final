## Build & Start

### Dependency

- Any JDK 1.8+
- Maven manages the rest of them

### Howto

- Build

```shell
./mvnw clean package -DskipTests=true
```

- Start

Exec this: `java -jar target/finalprj-0.0.1.jar`

Then goto any web browser, view <http://127.0.0.1:10088>.
